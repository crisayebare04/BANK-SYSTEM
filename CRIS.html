      # First Bank Uganda Account Opening App
      2+
      3+
      This project is a desktop Swing application for capturing new First Bank Uganda account opening requests. The interface includes validation, a summary panel, and a custom header with an image.
      4+
      5+
      ## Features
      6+
      7+
      - Required fields for applicant identity, contact details, date of birth, account type, branch, PIN, and opening deposit.
      8+
      - Date-of-birth selection using year/month/day combo boxes.
      9+
      - Inline validation messages for missing or invalid entries.
      10+
      - Summary panel that shows the saved application details.
      11+
      - Object-oriented account model with different account types.
      12+
      - CSV-based persistence so records can be opened in Excel.
      13+
      14+
      ## Project structure
      15+
      16+
      - `src/firstbank/` — Java source files for the GUI and business logic.
      17+
      - `out/` — compiled class files.
      18+
      - `lib/` — optional external libraries (currently not required for the CSV version).
      19+
      - `run-app.bat` — script to compile and launch the app.
      20+
      21+
      ## Requirements
      22+
      23+
      - Java JDK 17 or newer
      24+
      - A Windows terminal (PowerShell or Command Prompt)
      25+
      - Optional: Microsoft Excel to open the generated CSV file
      26+
      27+
      ## Setup
      28+
      29+
      1. Open a terminal in the project root.
      30+
      2. Make sure the project folder contains the `src` and `run-app.bat` files.
      31+
      3. If you want to use the generated CSV file in Excel, keep the project folder accessible and run the app from the project root.
      32+
      33+
      ## Compile the application
      34+
      35+

      36+
      37+

      ```powershell
      39+
      ```

      41+

      43+
      You can start the GUI using the batch script:
      44+
      45+
      ```powershell
      46+
      .\run-app.bat
      47+
      ```
      48+
      49+
      Or run it manually with:

      ```powershell
      52+
      java -cp out firstbank.FirstBankUgandaApp
      53+
      ```
      54+
      55+
      ## How it works
      56+
      57+
      - The form validates the user inputs before submission.
      58+
      - When the submit button is pressed, the application saves a record to:
      59+
      - `FirstBankUgandaApplications.csv`
      60+
      - The saved CSV file can be opened directly with Excel.
      61+
      62+
      ## Notes
      63+
      64+

      65+

      67+

      68+

      - If the CSV file is not created, check that the app has permission to write in the project folder.
